# Citadel Paint Ranges

**Epic:** Paint Information
**Type:** Reference
**Status:** Draft
**Source:** Games Workshop product catalog (warhammer-color.html)

## Summary

Comprehensive catalog of Games Workshop Citadel paint ranges extracted from the official Warhammer product page. Covers eight product lines across 331 paints: Base (57), Layer (93), Contrast (61), Shade (19), Air (45), Dry (24), Spray (11), and Technical (21). Citadel paints use names as their primary identifier rather than reference numbers, and the system is designed around a Base → Shade → Layer → Highlight workflow, with Contrast as a separate one-coat system.

## Brand Metadata Notes

Citadel paints use paint names as the primary identifier — there is no reference number system like Vallejo (70.xxx) or Army Painter (WP1xxx). The ranges are distinguished by paint type and pot style.

| Product Line | Paint Count | Pot Style    | Notes                                               |
| ------------ | ----------- | ------------ | --------------------------------------------------- |
| Base         | 57          | Standard pot | High pigment, full coverage foundation colors       |
| Layer        | 93          | Standard pot | Thinner formula for layering and highlighting       |
| Contrast     | 61          | Tall pot     | One-coat shade-and-highlight system                 |
| Shade        | 19          | Tall pot     | Thin washes for recesses and panel lines            |
| Air          | 45          | Standard pot | Pre-thinned for airbrush, mirrors Base/Layer colors |
| Dry          | 24          | Wide pot     | Thick chalky formula for drybrushing                |
| Spray        | 11          | Rattle can   | Primers and base coat sprays                        |
| Technical    | 21          | Standard pot | Texture pastes, blood, corrosion, varnish, medium   |
| **Total**    | **331**     |              |                                                     |

### Key Differences from Other Brands

- **No reference numbers:** Paint names are the sole identifier, unlike Vallejo (ref codes) or Army Painter (WPxxxx codes).
- **Shared names across ranges:** Many paints appear in multiple ranges with the same name but different formulations (e.g., Dawnstone exists as Layer, Dry, and Air; Mephiston Red exists as Base, Spray, and Air).
- **Workflow-oriented system:** Ranges are organized around the Base → Shade → Layer → Highlight painting workflow rather than by color family.
- **Contrast as parallel system:** Contrast paints provide an alternative one-coat workflow over light primer, separate from the traditional layering system.
- **Conversion chart overlap:** Some Citadel paint names appear as equivalents in Vallejo Game Color and Xpress Color conversion charts (e.g., White Scar ↔ 72.001 Dead White, Nuln Oil ↔ 73.201 Black).
