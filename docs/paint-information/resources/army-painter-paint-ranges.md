## Summary

Comprehensive catalog of Army Painter paint ranges extracted from official color chart and conversion PDFs. Covers three product lines: Warpaints Fanatic (180+ standard colors, 18 metallics, 19 effects/auxiliary, 18 washes), Speedpaint 2.0 (80+ colors including metallics and pastels), and Original Warpaints (legacy range with conversion data). Includes the Flexible Triad System for building Shadow/Base/Highlight combinations.

## Warpaints Fanatic

The current mainline acrylic paint range from Army Painter, replacing the original Warpaints. Dropper bottle format with improved pigment density and flow. Colors are organized into color families spanning the full hue wheel, with a Practical Naming Chart that assigns each paint a descriptive color name and a Conversion Chart mapping to Citadel equivalents.

## Flexible Triad System

The Fanatic range is organized into a Flexible Triad System where paints are arranged in rows by color family, ordered from dark (left) to light (right). The system is "flexible" because you can pick any 3 adjacent colors in a row to form a Shadow / Base / Highlight triad, rather than being locked into fixed triads.

Each row below lists paints dark-to-light. Two color families share each row, separated by an arrow, showing how the hue transitions smoothly across related families.

## Speedpaint 2.0

One-coat contrast-style paints designed for fast painting over white or light primer. The 2.0 formula fixes the reactivation issues of the original Speedpaint line. Apply in a single coat and the paint self-shades by flowing into recesses while leaving raised areas lighter. Includes standard colors, metallics, and pastels.

## Brand Metadata Notes

Currently the app tracks Army Painter as a single brand. The PDF data reveals multiple distinct product lines that could be represented:

| Product Line             | Paint Types                                          | Count                                              |
| ------------------------ | ---------------------------------------------------- | -------------------------------------------------- |
| Warpaints Fanatic        | Standard, Metallic, Effects, Auxiliary, Glow Effects | ~180 standard + 18 metallic + 19 effects/auxiliary |
| Warpaints Fanatic Washes | Wash                                                 | 18                                                 |
| Speedpaint 2.0           | Standard, Pastel, Metallic                           | ~73 standard + 6 pastel + 10 metallic              |
| Original Warpaints       | Legacy (discontinued)                                | —                                                  |

### Key Differences from Other Brands

- **No reference numbers:** Unlike Vallejo (which uses ref numbers like 72.001) or Citadel, Army Painter paints are identified solely by name. This means the paint name is the primary key in any data model.
- **Flexible Triad System:** Rather than fixed triads (like Vallejo's BSL system), Army Painter arranges colors in continuous rows where any 3 adjacent paints form a valid Shadow/Base/Highlight combination. This gives painters more freedom but makes the system harder to represent as discrete triad groups.
- **Practical Naming Chart:** Each Fanatic paint has both a creative name (e.g., "Turquoise Siren") and a descriptive color name (e.g., "Vivid Turquoise"). The descriptive name is useful for programmatic color matching and search.
- **Speedpaint vs Fanatic:** Speedpaints are a fundamentally different paint type (contrast/one-coat) from Fanatic (traditional layering paints). They should be tracked as separate sub-brands or paint types in the app.

### Conversion Chart Coverage

The Fanatic-to-Citadel conversion chart covers roughly 60% of Fanatic colors. Notable gaps include most of the desaturated cool greens, vibrant greens, teals, turquoises, magentas, and warm skin tones, which have no direct Citadel equivalent. This is expected as Army Painter's Fanatic range intentionally includes more nuanced color steps than Citadel's range.
