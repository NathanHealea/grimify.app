**Epic:** Paint Information

## Summary

Green Stuff World (GSW) is a smaller hobby brand with 32 acrylic paints currently tracked in the app. Unlike Citadel (331 paints across 8 ranges) or Vallejo (220+ Model Color refs alone), GSW's catalog in the app is limited to a single Acrylic line. All 32 hex values are estimated — there is no verified community database or official digital color chart available for cross-referencing.

## Brand Metadata Notes

Green Stuff World is a smaller hobby brand compared to the major manufacturers (Citadel, Vallejo, Army Painter). The app currently tracks only one paint type — Acrylic — with 32 colors.

### Key Differences from Other Brands

- **No reference numbers:** Like Citadel and Army Painter, GSW paints are identified solely by name — there is no reference number system like Vallejo (70.xxx).
- **Single paint type:** Only standard Acrylics are currently tracked. Citadel has 8 ranges, Army Painter has Fanatic + Speedpaint + Washes, and Vallejo has Model Color + Game Color + Xpress Color + Washes.
- **All hex values estimated:** None of the 32 hex values have been verified against a community database or official source. Other brands have partial or full verification from official color charts or community-maintained databases.
- **Smallest catalog:** At 32 paints, GSW is the smallest brand in the app by a significant margin.

### Expansion Opportunities

GSW manufactures several additional paint lines not yet cataloged in the app:

- **Dipping Inks** — Contrast-style inks similar to Citadel Contrast or Army Painter Speedpaint
- **Metallic Paints** — Dedicated metallic range
- **Fluorescent Paints** — UV-reactive and high-visibility colors
- **Chameleon Paints** — Color-shifting paints that change hue depending on viewing angle

Adding these ranges would bring GSW's catalog closer to parity with the other brands in the app.
