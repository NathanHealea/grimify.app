## Estimation Methodology

### Translucent paints (Contrast, Speedpaint, Shade, Wash, Ink, Xpress Color)

These paint types are inherently translucent — their appearance varies significantly depending on the primer color underneath. Hex values in this dataset represent the **approximate color when applied over a white or light primer**. This is an accepted limitation documented here for transparency.

### Metallic paints (Metallic, Liquid Metal)

Metallic paints have reflective properties that cannot be captured by a single hex value. Hex values represent the **base hue** of the metallic (e.g., gold → warm yellow-brown, copper → orange-brown, silver → neutral grey). The actual appearance includes specular highlights and color shifts that are not represented.

### Name-based estimation

For paints without community hex data, hex values were estimated from:
1. **Paint name color matching** — e.g., "Dark Rust Wash" → rust brown tone
2. **Citadel equivalent lookup** — using documented cross-brand equivalents
3. **Color description parsing** — from manufacturer product descriptions
4. **Product line matching** — Air/Dry/Spray paints that share names with Base/Layer paints use the same hex

## General Notes

- Hex codes represent the **swatch color** as displayed on websites, not necessarily the exact dried paint appearance
- Metallic paints cannot be accurately represented by a single hex code due to their reflective properties
- Some paints may have been discontinued or renamed by manufacturers since these sources were last updated
- Data originally compiled 2026-03-03, expanded 2026-03-12
