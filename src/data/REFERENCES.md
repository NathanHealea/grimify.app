# Paint Data References

Sources used to compile the paint hex color data in this directory.

## Citadel (202 paints: 49 Base, 80 Layer, 8 Edge, 61 Contrast, 4 other)

### Base, Layer, Edge (141 paints)

**Primary source:** [List of Citadel paints - HandWiki](https://handwiki.org/wiki/List_of_Citadel_paints)
- Hex codes sourced from the Games Workshop website color swatches
- Metallic paint hex values are approximations (reflective properties cannot be fully represented by a single hex code)
- Note: Paints marked as "(Metal)" on GW's site have had the suffix removed in our data

**Supporting sources:**
- [Games Workshop - Citadel Paint Color Matching (Encycolorpedia)](https://encycolorpedia.com/paints/games-workshop-citadel) — 174 Citadel paint colors with hex conversions
- [Citadel Colour Official Site](https://citadelcolour.com/the-paint-range/)
- [Citadel Colour Search Tool](https://kalamitykas.github.io/citadelcolourfinder/)
- [List of Citadel Paints - Games Workshop Wiki (Fandom)](https://gamesworkshop.fandom.com/wiki/List_of_Citadel_paints)

### Contrast (61 paints — ESTIMATED hex values)

**Note:** Contrast paints are translucent — their appearance depends on the primer color underneath. Hex codes represent the **approximate color when applied over a white/light primer**. No community database provides verified hex codes for Contrast paints.

**Sources for paint names:**
- [Citadel Painting System (PaintPad)](https://paintpad.app/paints/citadel-painting-system) — complete list of all 61 Contrast paint names
- [Citadel Colour Official Site](https://citadelcolour.com/the-paint-range/)
- [Review: The 25 new Citadel Contrast Colours (Tale of Painters)](https://taleofpainters.com/2022/07/review-the-25-new-citadel-contrast-colours/)
- [Visual comparison of all Citadel Contrast, Speedpaint 2.0, etc. (Tale of Painters)](https://taleofpainters.com/2023/11/top-tip-visual-comparison-of-all-61-contrast-23-speedpaints-24-gsw-dipping-inks/)

## General Notes

- Hex codes represent the **swatch color** as displayed on websites, not necessarily the exact dried paint appearance
- Metallic paints cannot be accurately represented by a single hex code due to their reflective properties
- Some paints may have been discontinued or renamed by manufacturers since these sources were last updated
- Data was compiled on 2026-03-03
